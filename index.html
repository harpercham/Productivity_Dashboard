Baseline rule: Workfront Logic provides Start and Finish. Actual completion is matched from Daily Point Update by ST + Point No. The target S-curve uses the Start date from Workfront Logic; if Start is blank, it falls back to first actual point date. Use the dashboard selector to switch between Standard S-Curve and Historical Productivity. Forecast finish uses the last 7 productive working days. Required productivity uses a 6-day working calendar (Mon–Sat, Sunday excluded) to the planned finish date. Recovery planner shows current forecast, +1 existing rig forecast after 7 working days, +1 new rig forecast after 14 working days, and the latest decision date. +1 rig assumes productivity doubles from the recent average. Rig increase is only recommended when forecast delay exceeds 14 working days.
